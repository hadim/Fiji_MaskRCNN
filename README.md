# Fiji_MaskRCNN

A Fiji plugin for [Mask RCNN semantic segmentation](https://arxiv.org/abs/1703.06870). The models runs from [a matterport Mask RCNN Tensorflow implementation](https://github.com/matterport/Mask_RCNN).
